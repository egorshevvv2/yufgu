## Описание программы

`Currency Converter` — это графическое приложение (GUI) на языке Python, предназначенное для быстрой и точной конвертации валют. Программа использует актуальные курсы обмена валют, получая их в реальном времени через внешний сервис (ExchangeRate-API). Приложение также ведет историю всех ваших конвертаций, сохраняя её локально.

### Основные функции:

1.  **Конвертация в реальном времени:** Точный перевод из одной валюты в другую с использованием свежих курсов.
2.  **Защита от ошибок ввода:** Программа проверяет, чтобы введенная сумма была положительным числом, исключая случайные опечатки.
3.  **История операций:** Каждая успешная конвертация автоматически сохраняется с указанием даты, времени, исходной и целевой валют, а также точной суммы и результата.
4.  **Локальное хранение:** История сохраняется в файл `conversion_history.json` и доступна после перезапуска программы.

## Как получить API-ключ (Важно!)

Для работы программы требуется бесплатный ключ от сервиса ExchangeRate-API. Без него конвертация работать не будет.

1.  Перейдите на сайт: [https://www.exchangerate-api.com/](https://www.exchangerate-api.com/)
2.  Введите свой email в поле на главной странице и нажмите "Get Free Key!".
3.  Завершите регистрацию (потребуется придумать пароль).
4.  В личном кабинете на сайте вы увидите ваш API Key (длинная строка из букв и цифр).
5.  Откройте файл `currency_converter.py` в любом текстовом редакторе.
6.  Найдите строку 21: `self.api_key = "ВАШ_API_КЛЮЧ"`
7.  Замените текст `ВАШ_API_КЛЮЧ` на скопированный вами ключ (оставив кавычки).
8.  Сохраните файл.

## Инструкция по запуску

1.  Убедитесь, что у вас установлен **Python 3**.
2.  Программе требуется библиотека `requests` для работы с сетью. Откройте терминал (или командную строку) и установите её командой:
    ```bash
    pip install requests
    ```
3.  Откройте терминал в папке с файлом `currency_converter.py`.
4.  Запустите программу командой:
    ```bash
    python currency_converter.py
    ```

## Примеры использования (Тесты)

*   **Тест 1: Успешная конвертация.**
    Убедитесь, что вы вставили API-ключ. Введите сумму "150", выберите "Из валюты" -> USD, "В валюту" -> EUR. Нажмите "Конвертировать". В поле результата появится актуальная сумма в Евро, а запись об операции добавится в таблицу истории ниже.
*   **Тест 2: Защита от текста.**
    В поле суммы введите слово "сто". Нажмите "Конвертировать". Программа покажет окно с ошибкой "Сумма должна быть положительным числом".
*   **Тест 3: Защита от отрицательных чисел.**
    В поле суммы введите "-50". Нажмите "Конвертировать". Программа покажет ту же ошибку о необходимости положительного числа.
*   **Тест 4: Сохранение истории.**
    Сделайте несколько разных переводов. Закройте программу и запустите её снова. Вся история ваших переводов должна загрузиться в таблицу из файла JSON.
