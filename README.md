## Описание программы
`Currency Converter` — это графическое приложение (GUI) на Python для конвертации валют по актуальным курсам. Программа позволяет быстро пересчитывать суммы между разными валютами, просматривать историю операций и сохранять данные для последующего использования.
### Основной функционал:
1.  **Выбор валют:** Удобные выпадающие списки для выбора валюты «из» и «в» (поддерживаются USD, EUR, GBP, JPY, RUB, CNY и др.).
2.  **Ввод суммы:** Поле для ввода конвертируемой суммы с проверкой корректности данных.
3.  **Конвертация:** Кнопка для выполнения пересчёта по актуальным курсам, получаемым через внешний API.
4.  **История операций:** Наглядная таблица с историей всех выполненных конвертаций (исходная валюта, целевая валюта, сумма, результат).
5.  **Сохранение истории:** Все операции автоматически сохраняются в локальный файл `history.json` и восстанавливаются при следующем запуске программы.
6.  **Защита от ошибок:** Программа проверяет корректность ввода:
    *   Сумма должна быть положительным числом.
    *   Отображение ошибок API и сетевых проблем.
    *   Уведомления о некорректном вводе через всплывающие окна.

)
*   **Хранение данных:** JSON
*   **HTTP‑запросы:** `requests`
*   **Контроль версий:** Git
## Инструкция по запуску
1.  Убедитесь, что у вас установлен **Python 3** (с поддержкой `tkinter` и `requests`).
2.  Склонируйте репозиторий на свой компьютер или скачайте файлы проекта (`main.py
`, `README.md
`, `.gitignore`).
3.  Установите необходимые зависимости:
    ```bash
    pip install requests
    ```
4.  Получите **API‑ключ** на [exchangerate-api.com
](https://www.exchangerate-api.com/)
 и замените `YOUR_API_KEY` в коде файла `main.py
` на ваш ключ.
5.  Откройте терминал (командную строку) в папке с проектом.
6.  Выполните команду для запуска:
    ```bash
    python main.py
    ```
## Примеры использования
*   **Конвертация валюты:**
    1. В выпадающем списке «Из» выберите `USD`.
    2. В выпадающем списке «В» выберите `RUB`.
    3. В поле «Сумма» введите `100`.
    4. Нажмите кнопку «Конвертировать».
    5. В всплывающем окне отобразится результат, например: `100 USD = 9 500.00 RUB`. Операция добавится в таблицу истории.
*   **Просмотр истории:**
    Все выполненные конвертации отображаются в таблице внизу окна. Вы можете прокрутить список, чтобы увидеть предыдущие операции. Каждая запись содержит:
    *   Валюту «из».
    *   Валюту «в».
    *   Исходную сумму.
    *   Результат конвертации.
*   **Обработка ошибок:**
    *   При вводе отрицательного числа или нуля появится сообщение: «Сумма должна быть положительным числом».
    *   При вводе текста вместо числа появится сообщение: «Введите корректное число».
    *   При проблемах с подключением к API появится сообщение: «Не удалось получить курс обмена».
